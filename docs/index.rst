=======================================
ScreenPy Requests: A ScreenPy Extension
=======================================

Not sure what ScreenPy is?
Check out `ScreenPy's documentation <https://screenpy-docs.readthedocs.io/en/latest/>`__ first!

``screenpy_requests`` extends ScreenPy
by providing an Ability,
Actions,
and Questions
which interact with |Requests|.
This enables your Actors
to :class:`~screenpy_requests.abilities.MakeAPIRequests`!
